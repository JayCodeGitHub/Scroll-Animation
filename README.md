# Scroll Animation #

A simple site where I learned to use GSAP and ScrollTrigger

## Getting Started

#### Download Dependencies ####

```bash
npm install
```

#### Start Dev Server ####

```bash
npm start
```

#### Build App ####

```bash
npm run build
```
